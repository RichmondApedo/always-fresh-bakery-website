### Method 1: Upload via GitHub Website (No Git Installation Required)

1. **Create GitHub Account**: 
   - Go to [github.com](https://github.com) and sign up (it's free!)

2. **Create New Repository**:
   - Click the green "New" button
   - Repository name: `always-fresh-bakery-website`
   - Make sure it's set to "Public"
   - ‚úÖ **Do NOT** check "Add a README file" (we already have one)
   - Click "Create repository"

3. **Upload Your Files**:
   - On the new repository page, click "uploading an existing file"
   - Open your File Explorer and go to: `C:\Users\ROSE\always-fresh-bakery-website`
   - Select ALL files: `index.html`, `styles.css`, `script.js`, `README.md`
   - Drag and drop them into GitHub
   - In the "Commit changes" section at the bottom:
     - Title: "Initial Always Fresh Bakery website"
     - Description: "Professional website for Always Fresh Bakery"
   - Click "Commit changes"

4. **Enable GitHub Pages**:
   - In your repository, click "Settings" (top menu)
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "Deploy from a branch"
   - Branch: "main" (should be selected automatically)
   - Folder: "/ (root)"
   - Click "Save"

5. **üéâ Your Website is Live!**:
   - After 2-5 minutes, your website will be available at:
   - `https://[your-github-username].github.io/always-fresh-bakery-website/`
   - GitHub will show you the exact URL in the Pages settings
