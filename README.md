# Always Fresh Bakery Website

A modern, responsive website for Always Fresh Bakery located in Dome Conca, Accra, Ghana.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Product Showcase**: Features Sugar Bread, Butter Bread, and Wheat Bread
- **Contact Integration**: Direct phone calling and WhatsApp messaging
- **Google Maps**: Interactive map showing bakery location
- **Social Media**: Links to Facebook, Instagram, Twitter, and WhatsApp
- **Contact Form**: Sends messages directly via WhatsApp
- **Modern UI**: Clean, professional design with smooth animations
